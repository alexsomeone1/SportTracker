# Як налаштувати Firestore Rules

## Спосіб 1: Через навігацію (рекомендовано)

1. У лівому меню знайдіть **"Firestore Database"** (вже виділено)
2. Натисніть на **"Firestore Database"**
3. У верхній частині сторінки знайдіть вкладки: **"Data"**, **"Indexes"**, **"Rules"**
4. Натисніть на вкладку **"Rules"**

## Спосіб 2: Пряме посилання

Натисніть "Got it" у повідомленні, а потім перейдіть до:
https://console.firebase.google.com/project/sport-tracker-6011e/firestore/rules

## Після переходу до Rules:

1. Ви побачите редактор з поточними правилами (зазвичай там є правила за замовчуванням)
2. **Видаліть** всі поточні правила
3. **Вставте** ці правила:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/trainings/{trainingId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

4. Натисніть кнопку **"Publish"** (внизу або зверху редактора)

## Що роблять ці правила?

- ✅ Дозволяють користувачам читати та записувати свої тренування
- ✅ Перевіряють, що користувач авторизований
- ✅ Перевіряють, що користувач може працювати тільки зі своїми даними
