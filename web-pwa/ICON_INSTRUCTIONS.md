# Інструкція: Як додати іконку для PWA

## Де знаходиться іконка?

Іконку потрібно помістити в папку:
```
web-pwa/icons/
```

## Які файли потрібні?

1. **icon-192.png** - 192x192 пікселів (для логотипу на сторінці авторизації)
2. **icon-512.png** - 512x512 пікселів (для PWA маніфесту)

## Звідки взяти іконку?

### Варіант 1: З Android додатку

1. Відкрийте Android Studio
2. Перейдіть до `app/src/main/res/`
3. Знайдіть папку з іконкою (зазвичай `mipmap-xxxhdpi` або `drawable`)
4. Скопіюйте файл `ic_launcher.png` або `ic_launcher_foreground.png`
5. Перейменуйте на `icon-192.png` та `icon-512.png`
6. Помістіть обидва файли в `web-pwa/icons/`

### Варіант 2: Створити нову іконку

1. Створіть іконку розміром 512x512 пікселів
2. Збережіть як `icon-512.png`
3. Зменшіть до 192x192 пікселів та збережіть як `icon-192.png`
4. Помістіть обидва файли в `web-pwa/icons/`

### Варіант 3: Використати онлайн-генератор

1. Відкрийте https://realfavicongenerator.net/ або https://www.pwabuilder.com/imageGenerator
2. Завантажте вашу іконку
3. Завантажте згенеровані файли
4. Помістіть їх в `web-pwa/icons/`

## Після додавання іконки:

1. Перезавантажте сторінку в браузері
2. Логотип має з'явитися на сторінці авторизації
3. Іконка має відображатися при встановленні PWA на iPhone

## Структура папки:

```
web-pwa/
├── icons/
│   ├── icon-192.png  ← Тут має бути іконка 192x192
│   └── icon-512.png  ← Тут має бути іконка 512x512
├── index.html
└── ...
```
