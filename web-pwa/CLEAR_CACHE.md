# Як очистити кеш браузера для оновлення стилів

## Chrome/Edge:
1. Натисніть `Cmd + Shift + R` (Mac) або `Ctrl + Shift + R` (Windows)
2. Або: `Cmd + Option + I` → Network → встановіть "Disable cache" → оновіть сторінку

## Safari:
1. Натисніть `Cmd + Option + E` (очистити кеш)
2. Або: `Cmd + R` (звичайне оновлення)

## Firefox:
1. Натисніть `Cmd + Shift + R` (Mac) або `Ctrl + Shift + R` (Windows)

## Або через DevTools:
1. Відкрийте DevTools (`F12` або `Cmd + Option + I`)
2. Клікніть правою кнопкою на кнопку оновлення
3. Виберіть "Empty Cache and Hard Reload"

---

**Важливо**: Після змін у CSS завжди використовуйте жорстке оновлення (`Cmd + Shift + R`), щоб побачити зміни!
