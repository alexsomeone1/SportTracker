# Як примусово оновити сторінку та очистити кеш

## Метод 1: Жорстке оновлення
- **Mac**: `Cmd + Shift + R`
- **Windows/Linux**: `Ctrl + Shift + R`

## Метод 2: Через DevTools
1. Відкрийте DevTools (`F12` або `Cmd + Option + I`)
2. Клікніть **правою кнопкою** на кнопку оновлення (🔄)
3. Виберіть **"Empty Cache and Hard Reload"**

## Метод 3: Очищення кешу через DevTools
1. Відкрийте DevTools (`F12`)
2. Перейдіть на вкладку **Network**
3. Встановіть галочку **"Disable cache"**
4. Оновіть сторінку (`Cmd + R` або `F5`)

## Метод 4: Повне очищення кешу браузера
1. Відкрийте налаштування браузера
2. Знайдіть "Очистити дані браузера" / "Clear browsing data"
3. Виберіть "Кешовані зображення та файли" / "Cached images and files"
4. Натисніть "Очистити" / "Clear"

---

**Важливо**: Після змін у CSS/JS завжди використовуйте жорстке оновлення!
